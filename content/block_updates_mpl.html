Bloquear las actualizaciones


Procedimiento

Si la consola tiene Tiramisu o Aroma de arranque manual:

Asegúrate de tener la versión más actual de Aroma o la versión más actual de Tiramisu. Si tienes Aroma y Tiramisu debes tener ambos actualizados.
  • En este punto debes tener la Wii U apagada con la tarjeta SD introducida.
  • Enciende la consola y accede al menú de Wii U.
  • Abre Información sobre salud y seguridad y mantén presionado el botón START ⊕︀ del Wii U GamePad hasta que salga un mensaje de advertencia en amarillo o hasta que salga el menú Boot Selector.
  • Si aparece un mensaje de advertencia en amarillo, presiona el botón X para bloquear por completo las actualizaciones del sistema de la consola. Eso es todo, ya tienes bloqueadas las actualizaciones. Apaga la consola.
  • Si aparece el menú del Environment Loader selecciona Aroma o selecciona Tiramisu y presiona el botón A e inmediatamente después mantén presionado el botón START ⊕︀ del Wii U GamePad hasta que salga el mensaje de advertencia en amarillo de arriba o hasta que salga el menú Boot Selector.
  • Si aparece el Boot Selector, en la esquina inferior derecha se muestra el texto:
    • Updates not blocked! Hold ⊕︀ + ⊝ to delete Update folder cuando NO están bloqueadas las actualizaciones.
    • Updates blocked! Hold ⊕︀ + ⊝ to restore Update folder cuando SÍ están bloqueadas las actualizaciones.
  • Mantén presionados los botones START ⊕︀ y SELECT ⊝ del Wii U GamePad al mismo tiempo hasta que el mensaje de esquina inferior derecha cambie.
  • Debe quedar el mensaje Updates blocked! Hold ⊕︀ + ⊝ to restore Update folder para que las actualizaciones queden completamente bloqueadas.
  • Recuerda que presionando el botón Y configuras la opción predeterminada del Boot Selector.
  • Selecciona Wii U Menu y luego presiona el botón A para abrir el menú de Wii U.
  • En el menú de Wii U, apaga la consola.
