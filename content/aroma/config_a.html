• Si activaste el arranque automático del PayloadLoader: Enciende la consola y mantén presionado el botón X hasta que salga el Environment Loader.
  • Si tu consola tiene PayloadLoader de arranque manual: Enciende la consola, accede al menú de Wii U, abre Información sobre salud y seguridad y mantén presionado el botón X del GamePad para abrir el Environment Loader.
  • Navega por el menú usando las flechas del D-Pad del GamePad, selecciona aroma.
  • Presiona el botón Y para configurar Aroma como el entorno predeterminado.
  • Presiona el botón A para abrir Aroma.
  • Si aparece un mensaje de advertencia acerca del update folder quiere decir que tu consola no está bloqueando las actualizaciones del sistema apropiadamente.
  • Presiona el botón X para bloquear por completo las actualizaciones del sistema de la consola o presiona el botón B para no volver a ver el mensaje de advertencia.
  • Bloquear las actualizaciones del sistema de la consola no afecta poder instalar las actualizaciones de los juegos. Bloquear las actualizaciones no es obligatorio, pero se recomienda encarecidamente. Puedes desbloquear las actualizaciones del sistema de la consola en cualquier momento después de completar la instalación de Aroma. Consulta la información de aquí cuando quieras desbloquear las actualizaciones.
  • Ahora debes estar en el Boot Selector de Aroma, con la opción Wii U Menu seleccionada, presiona el botón Y para configurar dicha opción como predeterminada y luego presiona el botón A para abrir el menú de Wii U.
  • En este punto ya finalizaste la instalación de PayloadLoader, del Environment Loader y de Aroma, tu consola iniciará con Aroma de forma predeterminada.
  • Con el entorno de Aroma cargado el menú de Wii U puede no verse diferente al menú de Wii U normal.
  • ¡Importante! Abres el menú de plugins de Aroma presionado L + Abajo + SELECT ⊝.
  • En la siguiente página vamos a añadir aplicaciones, módulos y plugins que agregan funcionalidad a Aroma.
  • Toma nota de los cuadros de información de abajo para usar correctamente Aroma.
Con el entorno de Aroma cargado en la consola no se usa el Homebrew Launcher. Las aplicaciones homebrew compatibles con Aroma (aplicaciones WUHB) aparecen directamente en el menú de Wii U, actualmente no todas las aplicaciones están garantizadas para funcionar correctamente con Aroma.
Si activaste el arranque automático del PayloadLoader puedes abrir el Environment Loader si enciendes la consola y mantienes presionado el botón X.
Si activaste el arranque automático del PayloadLoader puedes abrir el menú Boot Selector si enciendes la consola y mantienes presionado el botón START ⊕︀.
Si activaste el arranque automático del PayloadLoader puedes impedir que cargue el Environment Loader y Aroma sacando la tarjeta SD de la consola antes de encenderla. De esta forma la consola iniciará sin modificaciones. La siguiente vez que enciendas la consola con la tarjeta SD con los archivos del Environment Loader y Aroma cargará estos normalmente.
