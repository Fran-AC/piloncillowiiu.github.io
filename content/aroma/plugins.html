Añadir plugins WPS a Aroma

Los plugins WPS (Wii U Plugin System) son pequeñas aplicaciones y funciones complementarias para el entorno de Aroma. Se colocan en la carpeta sd:/wiiu/environments/aroma/plugins, examina la ruta de carpetas en las imágenes de referencia.

Estructura

  • La estructura correcta es: sd:/wiiu/environments/aroma/plugins/PluginName.wps.
  • Es incorrecto: sd:/wiiu/environments/aroma/modules/PluginName.wps.

Procedimiento

Si recién descargaste alguno de los siguientes archivos no es necesario que lo vuelvas a descargar.
  • Si tienes encendida la consola apágala.
  • Saca la tarjeta SD de la Wii U y ponla en tu PC para colocar los archivos.
  • Descarga los siguientes plugins:
    • Descarga re_nfpii. Permite configurar y usar la emulación de figuras amiibo.
      • Únicamente necesitas el archivo re_nfpii.zip.
    • Descarga Inkay. Permite configurar y usar Pretendo Network. Usa el menú de plugins de Aroma para activar o desactivar la conexión con los servidores de Pretendo Network.
      • Únicamente necesitas el archivo Inkay-pretendo.wps.
Si la PC te pregunta si deseas combinar carpetas selecciona que sí quieres combinar las carpetas.
Si la PC te pregunta si deseas reemplazar archivos selecciona que sí quieres reemplazar los archivos.
  • Descomprime el archivo re_nfpii.zip y entra en la carpeta plugins.
  • Copia el archivo re_nfpii.wps y pégalo en la carpeta sd:/wiiu/environments/aroma/plugins.
  • Así debe quedar el plugin re_nfpii en la tarjeta SD:
  • Es correcto: sd:/wiiu/environments/aroma/plugins/re_nfpii.wps.
  • Es incorrecto: sd:/wiiu/environments/aroma/modules/re_nfpii.wps.
Para poder usar el plugin de re_nfpii en la Wii U es obligatorio también colocar el módulo de re_nfpii.
  • Copia el archivo Inkay-pretendo.wps y pégalo en la carpeta sd:/wiiu/environments/aroma/plugins.
  • Así debe quedar el plugin Inkay en la tarjeta SD:
  • Es correcto: sd:/wiiu/environments/aroma/plugins/Inkay-pretendo.wps.
  • Es incorrecto: sd:/wiiu/environments/aroma/modules/Inkay-pretendo.wps.
  • Extrae la tarjeta SD de forma segura del dispositivo que usaste para hacer la copia de archivos e introdúcela en la ranura de la Wii U con la consola apagada.
Para poder jugar online usando Pretendo Network es necesario crear una nueva cuenta de usuario en la Wii U ya estando conectado a Pretendo Network para crear o vincular un "Identificador de Pretendo Network" para ese nuevo usuario de la consola. La Wii U en sí misma no puede diferenciar entre un "Identificador de Nintendo Network" (NNID) y un "Identificador de Pretendo Network" (PNID), pero el servicio de Nintendo Network y el servicio de Pretendo Network sí distinguen entre una cuenta de usuario con un NNID y otra cuenta de usuario con un PNID. Más información en el sitio oficial de Pretendo Network.
